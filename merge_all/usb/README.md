## 执行程序

1. 执行./ins.sh安装驱动

2. 执行./app执行测试程序

3. 运行结束后执行./rm.sh卸载驱动

***

## 功能

### 1. 智能卡

机器读取到智能卡时，会亮起LED0，反之，会关闭LED0。

### 2. 红外遥控

按键1 -- LED灯开启/关闭

按键2 -- 点阵显示模式更改

### 3. 键盘

发现键盘不灵敏时，请按键时稍稍用力

键盘第一行从左到右编号为0～3，以此类推，最后一行编号为12～15

按键0 -- LED0（因为LED0被智能卡线程控制，所以只会亮一下

按键1 -- LED1开启/关闭

按键2 -- LED2开启/关闭

按键3 -- 数码管显示温度

按键4 -- 数码管显示湿度

按键5 -- 蜂鸣器开启/关闭

按键6 -- 电机等待命令

按键7 -- 电机正向旋转

按键8 -- 电机反向旋转

按键9 -- 电机停止

按键15 -- 关闭程序
